Pasting from an external application:

1) Make sure you are in insert mode
2) Press Ctrl + Shift + v

Setting the line number by default:
vim ~/.vimrc
set number

Undo:
esc + u

Delete all text:
:1,$d

Go to a certain line:
<type line number>G

Typing Ctrl + S:
stops input - Press Ctrl + q to start typing again
